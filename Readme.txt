# website

This website is built as an assignment to an authentication page working with angular and node.js. The app contains 3 views-
1.Home page view- Public
2.Login page view- Public
3.Logged-in user view - Private

Home page consists of a welcome message and a status indicator whether you are logged in or not.
Login page consists of a username input, password input and a submit button.
Logged-in user consists of logout button to logout.


## Runnning the app
to run and watch the app 'nodemon ./bin/www' or you could type:

```bash
npm start
```

## Watching the frontend for any changes

```bash
npm run build
```

Hard-Coded Usernames and Passwords are in the users.json file.
